## プログラムは宣言たちからなる

まず、実際に動作する完全な C++ のソースコードを一つ示す。

```cpp
#include <print>

int add(int a, int b) {
  return a + b;
}

int main() {
  int result = add(2, 3);
  std::println("{}", result);
}
```

このプログラムの働きを先に述べると、`2` と `3` という整数に対し加算を行いその結果である `5` を出力する、というものである。
詳細は今後解説していくが、ここで一つ問いを立てたい。プログラムとは何から成り立つのか？
本書における見方は、プログラムは宣言たちからなる、というものである。
