---
title: "論理演算"
---

条件分岐の章で、比較演算の結果は真偽値(`bool` 型の値)だと見た。基本型の章では、`bool` 型の値どうしを組み合わせる演算は次章で扱うとした。本章でそれを扱う。

これまでに見た演算は、整数や実数に対する算術演算(`+`・`-`・`*`・`/`)と、2つの値を比べる比較演算(`<`・`==` など)だった。本章で見る **論理演算** は、`bool` 型の値どうしを組み合わせて、新しい真偽値を得る演算である。

## かつ `&&`

「`x` が 1 以上、かつ 10 以下」という条件を考える。これを数学のように `1 <= x <= 10` と書いても、意図した意味にはならない。これは `(1 <= x) <= 10` と解釈され、`1 <= x` の結果である真偽値を `10` と比べることになってしまう。

2つの条件を「かつ」でつなぐには、論理演算子 `&&` を使う。

```cpp
int x = 5;
bool in_range = 1 <= x && x <= 10;
```

`&&` は、左右の `bool` 型の値がともに `true` のときだけ `true` になり、それ以外は `false` になる。`x` が `5` のとき、`1 <= x` も `x <= 10` も `true` なので、`in_range` は `true` になる。

## または `||`

どちらか一方でも成り立てばよい、という条件には `||` を使う。

```cpp
bool edge = x == 1 || x == 10;
```

`||` は、左右のどちらかが `true` なら `true` になり、両方とも `false` のときだけ `false` になる。

## でない `!`

`!` は、1つの `bool` 型の値の真偽を反転する。`true` なら `false` に、`false` なら `true` になる。

```cpp
bool out_of_range = !(1 <= x && x <= 10);
```

`1 <= x && x <= 10` が `true` のとき、`out_of_range` は `false` になる。

## 左から評価し、決まれば右を見ない

`&&` と `||` は、まず左の式を評価する。`&&` で左が `false` なら、右を評価するまでもなく全体は `false` に定まるので、右の式は評価されない。`||` で左が `true` なら、同様に右は評価されない。左の結果だけで全体が決まる場合、右の式は実行されない。

## 条件を組み合わせる

論理演算によって、条件分岐の章で見た `if` の条件を、より豊かに書けるようになる。

```cpp
if (1 <= x && x <= 10)
{
    std::println("範囲内");
}
```

`if` の条件は `bool` 型の式であった。比較演算で得た `bool` 型の値を、論理演算でさらに組み合わせても、全体はやはり `bool` 型の式である。型の上では、これまでと同じ枠の中にある。
