---
title: "その他の演算と複合代入"
---

整数の型の章で、`int` には代表的な演算(`+`・`-`・`*`・`/`・`%`)があると述べ、他の演算は必要になる章で扱うとした。本章で、その残りの演算(ビットに対する演算とシフト)を見て、続いて、演算と代入を組み合わせた **複合代入** を扱う。

## ビット演算とシフト

整数は、内部では2進法のビット(`0` と `1`)の並びで表される。そのビットに直接はたらく演算がある。

- `&`・`|`・`^`:2つの整数のビットごとの論理積・論理和・排他的論理和
- `~`:ビットの反転
- `<<`・`>>`:ビットを左・右へずらすシフト

```cpp
int a = 6;
int b = 3;
int c = a & b;
```

`6` はビットで `110`、`3` は `011` である。`a & b` は、両方が `1` の桁だけを `1` にするので、結果は `010`、すなわち `2` になる。

これらは、論理演算の `&&`・`||` とは別の演算である。`&&`・`||` は `bool` 値を組み合わせるのに対し、`&`・`|` は整数のビットごとに働く。

ビット演算とシフトは、ビット単位の細かい操作に使う。本書ではビット操作を主筋とはせず、こうした演算があることを示すにとどめる。

## 複合代入

演算と代入を組み合わせた書き方がある。

```cpp
int x = 5;
x += 3;
```

`x += 3` は `x = x + 3` の短縮で、`x` に `3` を足した結果を `x` に代入し直す。`x` は `8` になる。

算術やビット演算・シフトの演算子には、こうした `op=` の形がそれぞれある。算術の `+=`・`-=`・`*=`・`/=`・`%=`、ビット演算やシフトの `&=`・`|=`・`^=`・`<<=`・`>>=` である。いずれも `x op= y` が `x = x op y` の短縮にあたる。

1だけ増やす、または減らすには、`++`・`--` がある。変数の前に置く **前置**(`++i`)と、後ろに置く **後置**(`i++`)がある。

```cpp
int i = 0;
i++;
```

`i++` も `++i` も `i` を1増やす。`i = i + 1` や `i += 1` と同じく、`i` は `1` になる。1減らすには `i--` または `--i` を使う。

両者は、式として値を使うときに違いが出る。後置は増やす前の値を、前置は増やした後の値を、その式の値とする。

```cpp
int x = 5;
int a = x++;  // a は 5、x は 6
int y = 5;
int b = ++y;  // b は 6、y は 6
```

`i` を増やすだけで、その値を使わない場合は、前置と後置のどちらでも同じである。

繰り返しの章では、ループ変数の更新を `i = i + 1` と書いた。これは `i += 1` や `i++` と書ける。複合代入も `++` も、すでにある変数の値を書き換える操作(代入の章で見た代入)の一種であり、ループ変数の更新のように書き換えが要る場面で使う。
