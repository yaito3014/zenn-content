---
title: "break と continue"
---

繰り返しの章では、条件が成り立つ間、本体を繰り返した。本章では、その繰り返しを途中で抜ける **`break`** と、今回の本体の残りを飛ばして次の繰り返しに進む **`continue`** を扱う。

## break

`break` は、繰り返しを途中で抜ける。実行が `break` に達すると、その時点で繰り返しが終わり、残りの回は実行されない。

例として、1から20までで、最初に7で割り切れる数を出力し、そこで抜ける。

```cpp
for (int i = 1; i <= 20; i++)
{
    if (i % 7 == 0)
    {
        std::println("{}", i);
        break;
    }
}
```

`i` が `7` のとき `i % 7 == 0` が成り立ち、`7` を出力して `break` で繰り返しを抜ける。`break` がなければ `14` も出力されるが、`break` があるので最初の1つで繰り返しが終わる。

## continue

`continue` は、今回の本体の残りを飛ばして、次の繰り返しに進む。

1から10までのうち、奇数だけを出力する。

```cpp
for (int i = 1; i <= 10; i++)
{
    if (i % 2 == 0)
    {
        continue;
    }
    std::println("{}", i);
}
```

`i` が偶数のとき、`continue` で本体の残り(`std::println`)を飛ばし、次の `i` に進む。出力は `1`、`3`、`5`、`7`、`9` になる。`for` の更新 `i++` は、`continue` のあとも行われる。

`while` には `for` のような更新の部分がないので、`continue` は条件の判定へ直接戻る。本体の中で値を更新している場合、`continue` でそこを飛ばすと、繰り返しが終わらなくなることがある。繰り返しの章で見た、終わらない繰り返しである。
